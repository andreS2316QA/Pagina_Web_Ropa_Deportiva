SportZone – Ropa Deportiva 👟🔥

SportZone es una pagina moderna y minimalista para una tienda de ropa deportiva de alto rendimiento. El proyecto se enfoca en una experiencia de usuario fluida, con un diseño oscuro (Dark Mode) que resalta la energía y el dinamismo del deporte a través de acentos en color rojo vibrante.
✨ Características Principales

    Diseño Dark Mode: Estética agresiva  utilizando una paleta de negros, grises y rojos.

    Sección Hero de Impacto: Uso de efectos de desenfoque (blur) en el fondo para priorizar la legibilidad del mensaje principal.

    Galería de Categorías: Tarjetas interactivas con efectos de zoom al pasar el cursor (hover).

    Grid de Productos: Exhibición limpia de artículos con etiquetas de "Nuevo" y comparativa de precios (ofertas).

    Interfaz Adaptable: Navegación optimizada mediante el uso de Flexbox para una estructura organizada.

    Sección de Valor: Iconografía para resaltar beneficios como envíos gratis y pagos seguros.

🛠️ Tecnologías Utilizadas

Para este proyecto se utilizaron estándares web modernos sin dependencias externas pesadas para garantizar la máxima velocidad de carga:

    HTML5: Estructura semántica para mejorar el SEO y la accesibilidad.

    CSS3: * Flexbox: Para la alineación y distribución del contenido.

        Pseudo-elementos (::before): Aplicados en la sección Hero para lograr el efecto de fondo desenfocado.

        Transiciones: Efectos suaves en botones e imágenes.

    Google Fonts / Web Safe Fonts: Tipografía limpia y legible.
